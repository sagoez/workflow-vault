# Workflow Vault

A curated collection of 90+ reusable shell workflows.

Browse, search, and copy resolved commands from the web — or use the [CLI](https://github.com/sagoez/workflow).

**[workflow.sagoez.com](https://workflow.sagoez.com)**

## Development

```bash
npm install
npm run dev      # dev server with hot reload on localhost:3000
npm run build    # build to web/site/index.html
```
